All Quick Wins in the TDI Learning Hub are **downloadable PDF documents**. No video content type is needed for quick wins. The `download_url` column in `hub_quick_wins` is the primary action.

**Why:** The Hub's quick wins are practical tools teachers can grab and use immediately. They're all PDFs (worksheets, checklists, reference guides, etc.).

**How to apply:** When building Quick Win detail pages, prioritize the download experience. The main CTA should always be "Download this tool" linking to download_url. Video player scaffolding can be removed. Community engagement features (sharing how you used it, reflections) need functional submit buttons, not placeholders.
