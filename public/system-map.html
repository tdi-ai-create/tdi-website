Data Flows -- How Information Moves

New Substack sub --> Hub account Weekly sync (Chris building by June 6) or self-signup via /hub
Hub upgrade click --> Stripe checkout /api/stripe/checkout creates session, redirects to Stripe, returns to /hub/membership?success=true
Stripe payment --> Hub membership Webhook at /api/stripe/webhook creates hub_memberships row with correct tier
First Hub login --> Welcome email HubAuthGuard fires POST /api/hub/emails/welcome via Resend (idempotent)
Q&A reply posted --> Email notification qa-handler.ts sends reply notification to original poster via Resend
Desi question --> Live Hub data get_hub_content tool queries hub_courses/hub_quick_wins via Hub Supabase client
Visitor on tdi.thinkific.com --> Redirect to /hub Footer script shows full-page overlay with "Visit the New Learning Hub" CTA
Course card on homepage --> Hub courses /api/courses queries hub_courses via Hub Supabase client, links to /hub/courses/:slug
