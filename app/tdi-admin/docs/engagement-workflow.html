Hub Engagement Workflow

End-to-end pipeline for creating, generating, and validating engagement checks in Learning Hub lessons.

Last updated: July 14, 2026 | Created by Rae Hughart

The Big Picture

Lesson Created → Content Added → Auto-Generate Checks → QA Validates → Learner Sees Interleaved Flow

This system ensures every lesson has meaningful engagement -- comprehension checks, reflections, action steps, and checkpoints -- interleaved with content so educators actively learn instead of passively watching.

Three groups coordinate: Bella creates and manages course content, Jasmine (AI agent) generates engagement checks, and Julie Lynn (QA agent) validates density. Rae reviews and approves before publish.

Two Paths to Engagement Checks

Checks can be created manually through the admin UI or automatically via agents. Both paths end with the same result: engagement checks in hub_quiz_questions positioned between content sections.

What the Learner Sees

Once checks are in place and the lesson has body content, the learner sees an interleaved flow instead of "content then quiz":

Check TypeColorLearner Experience
Multiple Choice / True-FalseGoldPick an answer, see immediate feedback + explanation. No penalty for wrong.
ReflectionTealFree-text private response. "This is just for you." Min 50 characters.
Action StepGreen"I'll try this" button + social proof ("12 educators have tried this").
CheckpointPurpleKey takeaways list. "I've got it" marks lesson complete.

Minimum Density Requirements

Every lesson must have at minimum:
  • 2 comprehension checks (multiple choice or true/false)
  • 1 action step ("Try It" -- a specific classroom action)
  • 1 checkpoint at the end (key takeaways)

Reflections encouraged but not required. Julie Lynn validates before publish.

Design Principles (Non-Negotiable)

These apply to every engagement check, whether human-written or AI-generated:
  • No penalties. Wrong answers get explanations, not point deductions.
  • No artificial friction. Never lock lessons behind quizzes.
  • Respect their time. Checks take 15-30 seconds. Low character minimums.
  • Warm tone. "That's it. Nice work." -- not "Correct! +10 XP"
  • No emojis. Colored dots, gradients, icons only.
  • Test understanding, not recall. "Which approach would be most effective?" not "Which was mentioned in the video?"
