TDI Data Flow Map

How data moves between systems to create intelligence

Last updated: July 19, 2026

The Learning Hub is the central data source. Educator activity flows outward to Leadership (for principals), Sales (for leads), CMO (for marketing), Creator Studio (for content strategy), and Funding (for grant impact metrics).

Learning Hub

Central data source -- educator activity, engagement, PD tracking

Leadership

Partner dashboards, engagement tracking, KPIs

Sales

Pipeline, opportunities, deal management

CMO Dashboard

Marketing funnel, Substack, campaigns

Creator Studio

Content creators, production pipeline

Funding

Grant pursuits, narratives, follow-up

Operations

Contracts, billing, renewals
